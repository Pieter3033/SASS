# SASS - South African Survival Surge

SASS (South African Survival Surge) is a survival game set in a harsh and dynamic South African environment.
Players must gather resources, craft tools, build shelter, and survive against nature and other dangers.

## Prototype Status
Early development. Current prototype includes:
- Small grassy map
- Player movement (WASD)
- Choppable tree
- Axe in hand with basic chop animation

## Controls
- `WASD` Move
- Mouse drag Orbit camera

## Dev Setup
```bash
npm install
npm run dev
```

## Notes
This is an early prototype. Expect rapid iteration and changes.
