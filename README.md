# SASS – South African Survival Surge

SASS (South African Survival Surge) is a survival game set in a harsh and dynamic South African environment.  
Players must gather resources, craft tools, build shelter, and survive against nature and other dangers.

## Features
- Open world survival
- Resource gathering
- Crafting system
- Base building
- Day / Night cycle
- Survival mechanics (health, hunger, thirst)

## Goal
Survive as long as possible and adapt to the environment.

## Controls
WASD – Move  
Mouse – Look / Aim  
E – Interact  
I – Inventory  
Space – Jump  
Shift – Sprint
